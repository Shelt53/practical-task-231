# practical-task-231
Планировщик задач на языке прогммирования Python 
В рамках данной работы в качестве языка программирования был выбран Python. Для создания графического пользовательского интерфейса была выбрана библиотека Tkinter. В качестве формата хранения данных был выбран JSON. Для сборки готового приложения в автономный исполняемый файл использовался инструмент PyInstaller.
Возможности
- Добавление, редактирование и удаление задач
- Приоритеты: высокий, средний, низкий
- Категории: Общее, Работа, Личное, Учёба, Здоровье, Покупки
- Сроки выполнения с подсветкой просроченных
- Фильтрация: все, активные, выполненные
- Сортировка: по приоритету, по дате
